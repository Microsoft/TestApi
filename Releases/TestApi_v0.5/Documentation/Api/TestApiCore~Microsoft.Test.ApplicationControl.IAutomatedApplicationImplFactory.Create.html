IAutomatedApplicationImplFactory.Create Method

Factory method for creating the IAutomatedApplicationImpl instance to be used by AutomatedApplication.

Parameters

settings
The settings to be passed the the implementation instance.

The AppDomain to create the implementation in. This is intended for in-proc scenarios where the AutomatedApplication needs to create the proxy in a separate AppDomain.

Return Value

Returns the application implementation to be used by AutomatedApplication
