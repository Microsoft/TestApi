CommandLineParser.ParseArguments Method

Sets properties on an object from a series of key/value string arguments that are in the form "/PropertyName=Value", where the value is converted from a string into the property type.

public static void ParseArguments(
   this object valueToPopulate,
   this IEnumerable<string> args
);

Parameters

valueToPopulate
The object to set properties on.
args
The key/value arguments describing the property names and values to set.

Return Value

Indicates whether the properties were successfully set. Reasons for failure reasons include a property name that does not exist or a value that cannot be converted from a string.

Exceptions

Exception Type Condition
ArgumentException Thrown when one of the key/value strings cannot be parsed into a property.
