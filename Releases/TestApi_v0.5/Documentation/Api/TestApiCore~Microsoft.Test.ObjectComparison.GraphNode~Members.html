Public Instance Properties

Name A name to identify this node. When comparing, the left and right nodes are matched by name.
Children Gets the collection of child nodes to this node. The child nodes represent properties or fields on an object.
Parent Represents the immediate parent to this node. The parent node of the root of the graph is null.
ObjectValue Contains the value of the object represented by this node.
ObjectType Provides the System.Type of the object represented by this node. Returns null if the object value is null.
Depth Gets the depth of this node from the root. If the depth of the root node is 0, root.child is 1.
QualifiedName Gets the fully qualified name of this node If the root node has a child that is named child1 and the child has another child that is named child12, qualified name of child12 would be root.child1.child12.

Public Instance Methods

GetNodesInDepthFirstOrder Performs a depth-first traversal of the graph with this node as root, and provides the nodes visited, in that order.
