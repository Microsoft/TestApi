Classes

Class Description
Constraint(T) Represents a relationship between parameters and their values, or other constraints.
ConstraintExtensions Holds extension methods that construct if-then-else constraints.
Model Provides a general-purpose model that generates a Variation. See also Model(T).
Model(T) Contains all the parameters and constraints for the system under test, and produces a set of variations by using combinatorial testing techniques.
Parameter(T) Represents a single variable and its values in the model whose values are of the specified type. Combinations of these values are used in the combinatorial generation of variations by the Model(T).
ParameterAttribute Specifies an attribute to decorate the properties of the class that is used for generating model parameters.
ParameterBase Provides a base class for the functionality that all parameters must implement.
ParameterValue(T) Represents a single value in a parameter.
ParameterValueBase Represents a single value in a parameter.
Variation Represents a tuple that has a single value for every Parameter(T) in the Model(T). The Model produces these by using combinatorial testing techniques.
