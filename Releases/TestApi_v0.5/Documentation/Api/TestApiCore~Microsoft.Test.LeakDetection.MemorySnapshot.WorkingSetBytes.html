MemorySnapshot.WorkingSetBytes Property

The current size of the working set of the process. The working set is the set of memory pages recently touched by the threads in the process. If free memory in the computer is above a threshold, pages are left in the working set of a process even if they are not in use. When free memory falls below a threshold, pages are trimmed from working sets. If they are needed they will then be soft-faulted back into the working set before leaving main memory.
