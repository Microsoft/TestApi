Microsoft.Test.VariationGeneration Namespace

Namespace Hierarchy

Classes

Class Description
Constraint Represents a relationship between parameters, their values or other constraints.
Model Contains all the parameters and constraints for the system under test and produces a set of variations using combinatorial test techniques.
Parameter Represents a single variable and its values in the model. Combinations of these values are used in the combinatorial generation of variations by the Model.
Variation Represents a tuple with a single value for every Parameter in the Model. The Model produces these using combinatorial test techniques.
