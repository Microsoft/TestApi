Snapshot Methods

The methods of the Snapshot class are listed below. For a complete list of Snapshot class members, see the Snapshot Members topic.

Public Static Methods

FromFile Creates a Snapshot instance from data in the specified image file.
FromRectangle Creates a Snapshot instance populated with pixels sampled from the specified screen rectangle, from the context of the Desktop.
FromWindow Creates a Snapshot instance populated with pixels sampled from the rectangle of the specified window.

Public Instance Methods

And Modifies the current image to contain the result of a bitwise AND of this Snapshot and the mask. This technique can be used to remove data from an image. http://en.wikipedia.org/wiki/Bitmask#Image_masks
Clone Creates a deep-copied clone Snapshot with the same value as the existing instance.
CompareToOverloaded. Compares the current Snapshot instance to the specified Snapshot to produce a difference image. Note: This does not compare alpha channels.
Crop Creates a new Snapshot based on the cropped bounds of the current snapshot.
DrawLine Draw a vertical line from the bottom of the specified column up to the height given.
Or Modifies the current image to contain the result of a bitwise OR of this Snapshot and the mask. This technique can be used to merge data from two images. http://en.wikipedia.org/wiki/Bitmask#Image_masks
Resize Creates a new Snapshot of the specified size from the original using bilinear interpolation.
ToFile Writes the current Snapshot (at 32 bits per pixel) to a file.
