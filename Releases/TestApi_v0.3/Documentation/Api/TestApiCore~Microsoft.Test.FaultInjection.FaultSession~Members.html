FaultSession overview

Public Static Methods

ClearGlobalFault Destroys the global fault session.
SetGlobalFault Sets a global fault session. This enables fault injection for all .NET processes launched after the creation of the global session. This functionality is often useful when testing server .NET applications, where one application typically consists of and launches many processes.

Public Instance Constructors

FaultSession Constructor Initializes a new instance of the FaultSession class with the specified FaultRule objects.

Public Instance Properties

LogDirectory Directory for all log files written by applications launched by this session.

Public Instance Methods

FindRule Finds the rule corresponding to a specified method.
GetProcessStartInfo Creates a ProcessStartInfo with the appropriate environment variables set for fault injection.
NotifyRuleChanges Notifies all test applications that fault rules have changed.
