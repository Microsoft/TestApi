OutOfProcessSettings Class

An OutOfProcessSettings object can be passed to the AutomatedApplication.Start method to indicate an AutomatedApplication should be created in its own process. The settings describe how to create the process through a ProcessStartInfo object. Optionally, the settings can also specify delegates to be called when the AutomatedApplication's main window opens or when the process exits.
