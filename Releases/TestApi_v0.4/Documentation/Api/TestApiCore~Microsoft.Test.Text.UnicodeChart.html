UnicodeChart Enumeration

Specifies a Unicode character code chart.

public enum UnicodeChart

Remarks

The Unicode standard defines a number of different character subsets, which are called Unicode character code charts (or Unicode charts for short). These charts are available on http://unicode.org/charts. The charts divide and categorize all symbols available in the Unicode range (0x0000 - 0x10FFFF) according to their common characteristics.
