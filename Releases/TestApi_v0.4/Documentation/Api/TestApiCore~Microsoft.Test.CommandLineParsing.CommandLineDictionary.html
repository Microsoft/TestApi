CommandLineDictionary Class

A dictionary that is aware of command line input patterns. All lookups for keys ignore case.

Example

This is a sample for parsing the following command line: Test.exe /verbose /runId=10 

        CommandLineDictionary cld = CommandLineDictionary.FromArguments(args);
        bool verbose = cld.ContainsKey("verbose");
        int runId = Int32.Parse(cld["runId"]);

        // You can also explicitly provide key and value identifiers for the cases
