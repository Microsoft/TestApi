Thread Safety

Remarks

Note that this class is used as "a filter" when generating character strings with StringFactory. Upon instantiation, all properties except CultureInfo of a StringProperties object (which are all Nullables) have null values, which means that the object does not impose any filtering limitations on the generated strings.

Setting properties to non-null values means that the value of the property should be taken into account by StringFactory during string generation. For example, setting MaxNumberOfCodePoints to 10 means "generate strings with up to 10 code points".

Example

The following example demonstrates how to create a random string with 10 Unicode points from the Greek character set: 

 StringProperties sp = new StringProperties();
 sp.MinNumberOfCodePoints = sp.MaxNumberOfCodePoints = 10;
 sp.UnicodeRange = new UnicodeRange(UnicodeChart.Greek);

 string s = StringFactory.GenerateRandomString(sp, 1234);
