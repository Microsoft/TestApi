ParameterValue.Weight Property

A value indicating whether this value should be chosen more or less frequently. Larger values will be chosen more often. The default is 1.0.

public double Weight { public get; public set; }

Remarks

Weighting creates preferences for certain values. Due to the nature of the algorithm used, the actual weight value has no intrinsic meaning (weighting one value at 10.0 and the others at 1.0 doesn’t mean it will appear 10x more often). The primary goal of the algorithm is to cover all the combinations with fewest test cases possible which often runs counter to honoring the weight. Weight acts a tie breaker when candidate values cover the same number of combinations.
